## 第一章:程式設計基礎

### 1.1 什麼是程式設計?

程式設計是一種創造性的活動,透過撰寫程式碼來解決問題。就像寫作文一樣,我們用特定的語言(程式語言)來表達我們的想法。

**程式設計的三大要素:**

1. **輸入(Input)**: 程式接收的資料
2. **處理(Process)**: 對資料進行運算或操作
3. **輸出(Output)**: 程式產生的結果

### 1.2 第一個程式

讓我們來看一個簡單的 Python 程式:

\`\`\`python
# 這是一個簡單的問候程式
name = input("請輸入你的名字: ")
print(f"你好, {name}! 歡迎學習程式設計!")
\`\`\`

這個程式做了三件事:
- 要求使用者輸入名字
- 將名字儲存在變數中
- 顯示問候訊息

## 第二章:變數與資料型態

### 2.1 變數的概念

變數就像是一個**容器**,用來存放資料。想像你有一個標籤貼紙,你可以把它貼在不同的盒子上。

| 資料型態 | 說明 | 範例 |
|---------|------|------|
| 整數(int) | 沒有小數點的數字 | 1, 42, -10 |
| 浮點數(float) | 有小數點的數字 | 3.14, -0.5 |
| 字串(string) | 文字資料 | "Hello", "你好" |
| 布林值(bool) | 真或假 | True, False |

### 2.2 變數命名規則

> **重要提醒**: 好的變數名稱能讓程式更容易理解!

變數命名的黃金法則:
- ✅ 使用有意義的名稱: `student_name` 比 `x` 好
- ✅ 使用小寫字母和底線: `my_variable`
- ❌ 不要使用保留字: `if`, `for`, `while`
- ❌ 不要用數字開頭: `1st_name` 是錯的

## 第三章:控制流程

### 3.1 條件判斷

程式需要根據不同情況做出決定,這時候就需要 `if` 語句:

\`\`\`python
age = 18

if age >= 18:
    print("你已經成年了!")
else:
    print("你還未成年")
\`\`\`

### 3.2 迴圈

當我們需要重複執行某些動作時,可以使用迴圈:

\`\`\`python
# for 迴圈範例
for i in range(5):
    print(f"這是第 {i+1} 次")

# while 迴圈範例
count = 0
while count < 3:
    print("繼續執行...")
    count += 1
\`\`\`

## 第四章:函式

### 4.1 為什麼需要函式?

函式就像是**食譜**,把一系列步驟包裝起來,需要時就可以重複使用。

**函式的優點:**
- 🔄 程式碼可重複使用
- 📦 讓程式更有組織
- 🐛 更容易除錯
- 👥 方便團隊合作

### 4.2 定義函式

\`\`\`python
def greet(name, time="早安"):
    """
    這是一個問候函式
    參數:
        name: 要問候的人名
        time: 問候語(預設為"早安")
    """
    message = f"{time}, {name}!"
    return message

# 使用函式
result = greet("小明")
print(result)  # 輸出: 早安, 小明!
\`\`\`
