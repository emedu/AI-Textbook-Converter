# 🚀 一分鐘快速開始

> 最簡單的使用說明 - 只要 3 個步驟!

---

## 第一次使用

### 步驟 1: 安裝軟體(只需要做一次)

1. **安裝 Node.js**
   - 前往: https://nodejs.org/
   - 下載左邊的「LTS」版本
   - 安裝後重開機

2. **安裝 Pandoc**
   - 前往: https://pandoc.org/installing.html
   - 下載 Windows 版本
   - 安裝

3. **設定 API Key**
   - 用記事本打開 `.env` 檔案
   - 把您的 Gemini API Key 貼進去
   - 儲存

---

### 步驟 2: 安裝程式套件(只需要做一次)

1. 在專案資料夾上按右鍵
2. 選擇「在終端機中開啟」
3. 輸入: `npm install`
4. 等待 3-5 分鐘

---

### 步驟 3: 開始使用

1. **雙擊** `啟動程式.bat`
2. 瀏覽器會自動打開
3. 上傳 `.md` 檔案
4. 點擊「開始轉換」
5. 下載 PDF 和 Word 檔案

---

## 每次使用

只需要:
1. 雙擊 `啟動程式.bat`
2. 上傳檔案
3. 下載結果

就這麼簡單! 🎉
