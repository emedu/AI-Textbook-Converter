# 📚 AI 教材轉換系統

> 這是一個能把 Markdown 教材轉換成漂亮 PDF 和 Word 文件的工具，就像把筆記變成專業教科書一樣簡單！

---

## 📖 文件索引

本專案包含完整的文件說明，請依您的需求閱讀：

### 1. [使用者手冊 (User Manual)](./使用者手冊.md)
*   **適用對象**: 一般使用者
*   **內容**: 安裝教學、操作步驟、常見問題排解。
*   👉 **[點擊閱讀](./使用者手冊.md)**

### 2. [開發者文件 (Developer Guide)](./開發者文件.md)
*   **適用對象**: 程式開發人員、維護者
*   **內容**: 系統架構、程式碼邏輯、環境建置、擴充指南。
*   👉 **[點擊閱讀](./開發者文件.md)**

### 3. [專案開發紀錄與交接 (Handover Log)](./專案開發紀錄與交接事項.md)
*   **適用對象**: 專案接手人
*   **內容**: 技術決策原因、已知限制、未來優化方向。
*   👉 **[點擊閱讀](./專案開發紀錄與交接事項.md)**

---

## ⚡ 快速啟動

1. 雙擊 **`啟動程式.bat`**。
2. 瀏覽器自動開啟後，上傳 `.md` 檔案即可轉換。

---

## 📁 核心目錄結構

```
AI教材轉換專案115/
├── 📄 使用者手冊.md        ← 新手請看這
├── 📄 開發者文件.md        ← 工程師請看這
├── 📄 啟動程式.bat         ← 點兩下開始用
├── 📁 src/                ← 核心程式碼
└── 📁 output/             ← 產出的檔案在這裡
```
