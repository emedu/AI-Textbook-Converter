# 📚 AI 教材轉換系統 - 使用說明

> 這是一個能把 Markdown 教材轉換成漂亮 PDF 和 Word 文件的工具
> 就像把筆記變成專業教科書一樣簡單!

## 🎯 這個程式能做什麼?

- ✅ 上傳 Markdown 檔案(.md)
- ✅ 自動產生專業排版的 PDF(有頁碼、目錄、頁眉頁腳)
- ✅ 同時產生 Word 檔案(.docx)
- ✅ 使用 AI 自動整理格式(未來功能)

---

## 📦 第一次使用 - 安裝步驟

### 步驟 1: 安裝 Node.js

1. 打開瀏覽器,前往: https://nodejs.org/
2. 下載「LTS 版本」(推薦版本)
3. 安裝時一直按「下一步」就好
4. 安裝完成後,重新啟動電腦

### 步驟 2: 安裝 Pandoc(用來產生 Word)

1. 前往: https://pandoc.org/installing.html
2. 下載 Windows 版本
3. 安裝時一直按「下一步」

### 步驟 3: 設定 API Key

1. 打開資料夾中的 `.env` 檔案(用記事本打開)
2. 找到這一行: `GEMINI_API_KEY=請把你的API金鑰貼在這裡`
3. 把 `請把你的API金鑰貼在這裡` 替換成您的 Gemini API Key
4. 儲存檔案

### 步驟 4: 安裝程式需要的套件

1. 在這個資料夾上按右鍵
2. 選擇「在終端機中開啟」
3. 輸入: `npm install`
4. 按 Enter,等待安裝完成(可能需要 3-5 分鐘)

---

## 🚀 如何使用

### 啟動程式

1. 雙擊 `啟動程式.bat` 檔案
2. 會自動打開瀏覽器
3. 看到上傳頁面就成功了!

### 轉換教材

1. 點擊「選擇檔案」按鈕
2. 選擇您的 Markdown 檔案(.md)
3. 點擊「開始轉換」
4. 等待處理完成
5. 下載 PDF 和 Word 檔案

## ❓ 常見問題

### Q: 程式打不開怎麼辦?
A: 確認是否已安裝 Node.js,可以開啟終端機輸入 `node -v` 檢查

### Q: 轉換失敗怎麼辦?
A: 檢查 `.env` 檔案中的 API Key 是否正確

### Q: 可以給其他同事用嗎?
A: 可以!把整個資料夾複製到其他電腦,重複「第一次使用」的步驟即可

### Q: 需要網路嗎?
A: 需要。因為要呼叫 Gemini API 進行 AI 處理
