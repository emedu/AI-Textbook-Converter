# 📝 智能文字轉換功能說明

## 🎯 功能介紹

系統現在具備**智能文字格式偵測**功能！

當您上傳 `.md` 檔案時，系統會自動判斷：
- 如果是**標準 Markdown 格式**（有 `#` 標題符號）→ 直接處理
- 如果是**純文字格式**（沒有 Markdown 語法）→ **自動轉換**為 Markdown

## ✨ 自動轉換規則

系統會智能識別以下內容並自動加上 Markdown 標記：

### 1. 標題識別

| 原始文字 | 轉換結果 | 說明 |
|---------|---------|------|
| `目錄` | `# 目錄` | 關鍵字自動識別為 H1 |
| `導論` | `# 導論` | 關鍵字自動識別為 H1 |
| `第一章：xxx` | `# 第一章：xxx` | 章節標題為 H1 |
| `1. 標題文字` | `## 標題文字` | 一級編號為 H2 |
| `1.1 小標題` | `### 小標題` | 二級編號為 H3 |
| `1.1.1 細項` | `#### 細項` | 三級編號為 H4 |

### 2. 自動清理

系統會自動移除：
- 行尾的點點點（`...`、`……`）
- 頁碼數字（如 `.....23`）

### 3. 列表項目

如果您的文字有：
- `• 項目一`
- `1) 項目二`
- `- 項目三`

都會統一轉換為標準 Markdown 列表格式：
```markdown
- 項目一
- 項目二
- 項目三
```

## 📋 使用方式

**完全不需要任何額外操作！**

1. 將您的 Word/PDF 內容複製到記事本
2. 存成 `.md` 檔案
3. 直接上傳到系統

系統會自動偵測並轉換，您會得到排版漂亮的 PDF 和 Word 文件！

## 💡 提示

如果您想要更精確的控制，也可以手動加上 Markdown 語法：
- `#` = 大標題（H1）
- `##` = 中標題（H2）
- `###` = 小標題（H3）
- `-` = 列表項目
