# 🎓 快速開始指南 - 給完全不懂程式的人

## 📌 第一步:安裝必要軟體

### 1. 安裝 Node.js

**什麼是 Node.js?**
就像是這個程式的「引擎」,沒有它程式就跑不動。

**怎麼安裝?**
1. 打開瀏覽器
2. 前往: https://nodejs.org/
3. 看到兩個綠色按鈕,點擊左邊那個「LTS」版本
4. 下載完成後,雙擊安裝檔
5. 一直按「下一步」就好(不用改任何設定)
6. 安裝完成後,**重新啟動電腦**

**怎麼確認安裝成功?**
1. 按鍵盤的 `Win + R`
2. 輸入 `cmd` 然後按 Enter
3. 在黑色視窗中輸入: `node -v`
4. 如果看到類似 `v20.11.0` 的版本號,就是成功了!

---

### 2. 安裝 Pandoc(用來產生 Word 檔)

**什麼是 Pandoc?**
這是一個專門轉換文件格式的工具。

**怎麼安裝?**
1. 前往: https://pandoc.org/installing.html
2. 找到「Windows」那一行
3. 點擊下載連結
4. 下載完成後,雙擊安裝檔
5. 一直按「下一步」

**怎麼確認安裝成功?**
1. 打開 cmd(同上)
2. 輸入: `pandoc --version`
3. 看到版本號就成功了!

---

## 📌 第二步:設定 API Key

**什麼是 API Key?**
就像是一把「鑰匙」,讓程式可以使用 Google 的 AI 功能。

**怎麼設定?**
1. 在專案資料夾中找到 `.env` 這個檔案
2. 用「記事本」打開它(右鍵 → 開啟檔案 → 記事本)
3. 找到這一行:
   ```
   GEMINI_API_KEY=請把你的API金鑰貼在這裡
   ```
4. 把 `請把你的API金鑰貼在這裡` 刪掉
5. 貼上您的 Gemini API Key
6. 儲存檔案(Ctrl + S)

**範例:**
```
GEMINI_API_KEY=AIzaSyABC123def456GHI789jkl
```

---

## 📌 第三步:第一次啟動

**只需要做一次!**

1. 在專案資料夾上按右鍵
2. 選擇「在終端機中開啟」(或「Open in Terminal」)
3. 輸入: `npm install`
4. 按 Enter
5. 等待 3-5 分鐘(會自動下載需要的套件)
6. 看到「added XXX packages」就完成了!

---

## 📌 第四步:使用程式

### 啟動程式

**超簡單!**
1. 找到資料夾中的 `啟動程式.bat` 檔案
2. 雙擊它
3. 會自動打開瀏覽器
4. 看到紫色漸層的上傳頁面就成功了!

### 轉換教材

1. **準備檔案**: 確保您的教材是 `.md` 格式(Markdown)
2. **上傳檔案**: 
   - 點擊網頁上的上傳區域
   - 或直接把檔案拖進去
3. **開始轉換**: 點擊「開始轉換」按鈕
4. **等待完成**: 看到進度條跑到 100%
5. **下載檔案**: 
   - 點擊「下載 PDF 檔案」
   - 點擊「下載 Word 檔案」

## ❓ 常見問題

### Q1: 雙擊 `啟動程式.bat` 沒反應?
**A:** 可能是 Node.js 沒裝好
- 重新安裝 Node.js
- 記得安裝完要重開機

### Q2: 轉換時出現錯誤?
**A:** 檢查以下項目:
- ✅ API Key 是否正確設定
- ✅ 網路是否連線
- ✅ 檔案是否為 .md 格式

### Q3: Word 檔案無法產生?
**A:** 可能是 Pandoc 沒裝好
- 重新安裝 Pandoc
- 確認安裝路徑有加入系統環境變數

### Q4: 如何關閉程式?
**A:** 
- 關閉瀏覽器視窗
- 在黑色的命令提示字元視窗按 `Ctrl + C`
- 或直接關閉黑色視窗

### Q5: 可以給同事用嗎?
**A:** 可以!
1. 把整個資料夾複製到同事的電腦
2. 他也要安裝 Node.js 和 Pandoc
3. 他也要設定自己的 API Key
4. 然後執行 `npm install`
5. 就可以用了!
