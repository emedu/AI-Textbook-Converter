# 🟢 免安裝版設定指南 (Portable Setup)

為了讓您不用在電腦上安裝軟體也能使用，我們採用「綠色免安裝」的方式。
請按照以下三個步驟，下載必要的工具放入 `tools` 資料夾即可。

## 步驟 1: 準備工具資料夾

確保您的專案資料夾結構如下 (我已經幫您建好空資料夾了)：

```
AI教材轉換專案115/
└── tools/
    ├── node/      (即將放入 Node.js)
    └── pandoc/    (即將放入 Pandoc)
```

## 步驟 2: 下載並放入 Node.js (程式引擎)

1. **下載**: [點擊下載 Node.js (免安裝版)](https://nodejs.org/dist/v20.11.0/node-v20.11.0-win-x64.zip)
2. **解壓縮**: 打開下載的 zip 檔。
3. **放入**: 將 zip 裡面的 **所有內容** 解壓縮到 `tools/node/` 資料夾中。
   - 正確路徑檢查: 您應該會在 `tools/node/` 裡面看到 `node.exe` 檔案。

## 步驟 3: 下載並放入 Pandoc (Word 轉換器)

1. **下載**: [點擊下載 Pandoc (免安裝版)](https://github.com/jgm/pandoc/releases/download/3.1.11.1/pandoc-3.1.11.1-windows-x86_64.zip)
2. **解壓縮**: 打開下載的 zip 檔。
3. **放入**: 將 zip 裡面的 **所有內容** 解壓縮到 `tools/pandoc/` 資料夾中。
   - 正確路徑檢查: 您應該會在 `tools/pandoc/` (這層可能還有一層資料夾) 裡面找到 `pandoc.exe`。
   - **注意**: 如果解壓縮後多了一層資料夾 (例如 `pandoc-3.1.11.1`), 請把裡面的東西拿出來，直接放在 `tools/pandoc/` 裡，讓 `pandoc.exe` 直接位於 `tools/pandoc/pandoc.exe` (或 `tools/pandoc/bin/pandoc.exe`)。
   - *我們的啟動程式會自動搜尋這兩個位置，您不用擔心。*

## 步驟 4: 完成！

現在，您只需要回到主目錄，雙擊 `啟動程式.bat` 即可開始使用！
第一次啟動時，它會自動進行初始化，請稍等它一下。
